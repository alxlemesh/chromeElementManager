# Quick Start Guide

## Install the Extension (2 minutes)

1. Open Chrome browser
2. Type `chrome://extensions/` in the address bar
3. Turn ON "Developer mode" (top right toggle)
4. Click "Load unpacked" button
5. Select this folder: `clickToRemoveHiglight`
6. Done! The extension icon will appear in your toolbar

## Key Features

- **In-Page Menu**: No popup - everything happens on the page
- **Live Preview**: See what element you're hovering over before clicking
- **Mode Indicator**: Always know what mode you're in
- **Persistent**: Changes saved forever (until you clear them)
- **Per-Page Lists**: Only shows elements from the current page
- **Easy Cleanup**: Click √ó next to items or use "Clear All"
